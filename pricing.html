LambdaX Pricing

Simple, predictable, and fair pricing. Our model is designed to be transparent, with no hidden fees.

Pricing at a Glance

Your monthly bill is made of three simple components:

  1. Compute Fee: You pay for the GPU resources you use, billed by the millisecond.
  2. Invocation Fee: A tiny flat fee for each time your function is run.
  3. Networking Add-ons (Optional): A flat monthly fee for advanced features that connect LambdaX securely to your private VPC.

Core Pricing: Compute & Invocations

The core of our pricing is the Slice-Second, a single, unified metric that combines vCPU, system memory (RAM), and GPU memory (VRAM) into one simple unit.

The LambdaX Slice

A 1x Slice is the smallest unit of compute capacity you can request.

1x Slice = 0.25 vCPU + 1 GB Host RAM + 1.5 GB VRAM

You configure your function with the number of slices needed to meet its peak requirement. For example, a function needing 3.5 GB of VRAM would be configured as a 3x Slice (which provides 4.5 GB of VRAM).

Usage-Based Fees

Component Price
Invocation Fee $0.20 per 1 million invocations
Compute Fee $0.00005 per Slice-Second

Formula: Total Compute Cost = (Number of Slices) x (Duration in Seconds) x $0.00005

Detailed Pricing Example

Let's walk through an example for a typical video processing function.

1. Function Requirements:

  • vCPU needed: 0.8
  • Host RAM needed: 3.5 GB
  • GPU VRAM needed: 2.0 GB

2. Choose Your Slice Size:

To meet the 3.5 GB Host RAM requirement, you would select a 4x Slice, which provides 4 GB of Host RAM, 1.0 vCPU, and 6.0 GB of VRAM.

3. Measure Duration:

Your function runs and takes 1.2 seconds to complete.

4. Calculate the Cost:

  • Slice-Seconds consumed: 4 Slices * 1.2 seconds = 4.8 Slice-Seconds
  • Cost for this invocation: 4.8 Slice-Seconds * $0.00005/Slice-Second = $0.00024

If you were to run this function 1 million times in a month, your total bill would be:

  • Invocation Cost: 1,000,000 * ($0.20 / 1,000,000) = $0.20
  • Compute Cost: 1,000,000 * $0.00024 = $240.00
  • Total Monthly Bill: $240.20

Optional Add-on: Advanced Networking

For functions that need to interact securely with your private cloud resources, we offer powerful networking features based on AWS PrivateLink, billed hourly. This ensures that traffic between your VPC and LambdaX never traverses the public internet.

Total Cost Transparency: Enabling these features requires provisioning resources in your own AWS account for maximum security and control. The table below shows both the LambdaX fee and the estimated, separate charge you will see on your AWS bill, prorated by the hour.

Why is there an AWS cost? The AWS cost pays for the dedicated networking endpoint (a VPC Endpoint or a Network Load Balancer) that runs securely inside your own AWS account. This gives you full control and ensures the connection is truly private. We provide simple CloudFormation templates to make this setup as easy as possible.
